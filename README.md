# Event Ticket Booking Platform

A full stack event ticket booking app where users can explore events, choose seats, and manage their bookings.

I built this project mainly to get hands-on experience with authentication, protected routes, booking workflows, and connecting frontend with backend APIs in a real-world style application.

## Live Demo

https://smart-event-ticketing.vercel.app/

# Features

- User login and signup with JWT authentication
- Browse and book events
- Seat selection workflow
- Organizer dashboard for managing events
- Booking history page
- Responsive interface for different screen sizes

# Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Context API
- Axios
- 
# Future Improvements
